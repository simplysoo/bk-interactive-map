# Интерактивная карта БК v0.1.11

Официальный релиз расширения для подземелий БК.

## Что внутри

- Синхронизация карты с текущей клеткой персонажа.
- Отображение направления взгляда.
- Поддержка движения клавишами.
- Продолжение синхронизации после боя.
- Посещенные клетки с красной рамкой.
- Маршруты к клеткам, монстрам, NPC, сундукам и тайникам.
- Списки монстров, сундуков, тайников, NPC и людей.
- Полный возможный дроп сундуков и тайников.
- Учет поднятого дропа из игровых сообщений.
- Масштаб, перетаскивание карты, режимы `Игрок` и `Вся карта`.
- Поддержка Грибницы и ППК.
- Добавлена Канализация: 1-3 этажи.
- Исправлено автоопределение пещеры: Канализация больше не переключается ошибочно на ППК.
- Усилено автоопределение всех пещер: Грибница, ППК и Канализация определяются только по уникальным названиям.
- Маршруты теперь учитывают телепорты и переходы внутри текущей карты.
- При выборе закрытой двери маршрут строится к клетке с нужным ключом.
- В памяти остается только текущая карта, без накопления карт других этажей.
- Исправлен выбор клетки с первого клика: маршрут строится сразу при обычном клике по карте, без повторных нажатий.
- Перетаскивание карты сохранено: короткий клик выбирает клетку, движение больше 6 px считается drag.
- В ППК баррикады больше не используются как телепорты при построении маршрута; маршрутные спец-переходы учитывают только рабочие порталы.
- Добавлена Бездна: 4 этажа карты, руководства, NPC, монстры, сундуки и тайники.
- Автоопределение Angels city: при URL `angelscity.combats.com` расширение автоматически выбирает карту `Бездна`.
- Добавлены все карты Катакомб: 1 этаж, 1 этаж (Провал), 2, 3 и 4 этажи, базовые руководства, NPC, монстры, сундуки и тайники.
- Исправлена сетка карточек монстров: длинные описания ботов больше не растягивают дроп и список.
- Добавлен поиск по монстрам в панели: фильтр работает по названию, клеткам и дропу.

## Установка

1. Скачать архив релиза с GitHub.
2. Распаковать папку расширения.
3. Открыть `chrome://extensions`.
4. Включить `Режим разработчика`.
5. Нажать `Загрузить распакованное расширение`.
6. Выбрать папку, где лежит `manifest.json`.
7. Перезагрузить вкладку игры.

## Обновление с прошлой версии

1. Заменить старую папку расширения новой.
2. Нажать `Обновить` на странице `chrome://extensions`.
3. Перезагрузить вкладку БК.
