# Интерактивная карта БК

Интерактивная карта БК - браузерное расширение для Chrome и Chromium-браузеров, которое добавляет в игру удобную карту подземелий с синхронизацией персонажа, маршрутами, объектами, NPC, монстрами и дропом.

Проект распространяется для личного использования игроками. Код не является open-source: перепубликация, продажа, удаление копирайтов, переименование проекта и выпуск сторонних сборок запрещены. Подробные условия указаны в [LICENSE.md](LICENSE.md).

## Возможности

- Синхронизация текущей клетки с персонажем в подземелье.
- Отслеживание обычных переходов, быстрых клавиш и движения после боя.
- Отображение направления взгляда по игровому тексту направления.
- Подсветка посещенных клеток красной рамкой.
- Маршрут до выбранной клетки, монстра, NPC, сундука или тайника.
- Перетаскивание карты мышью.
- Масштабирование карты с отображением текущего процента.
- Быстрые режимы масштаба: `Игрок` и `Вся карта`.
- Вкладка `Клетка` с подробностями выбранной клетки.
- Вкладка `Рыцарство` с важной информацией по подземелью.
- Вкладка `Дроп` с учетом поднятых предметов из игровых сообщений.
- Боковая панель объектов: монстры, сундуки и тайники, NPC и люди.
- Группировка одинаковых монстров по названию с количеством.
- Отдельный вывод сундуков и тайников без склейки дублей, чтобы было видно каждую точку.
- Полный возможный дроп для сундуков и тайников.
- Зачеркивание убитых монстров.
- Загрузка карт и справочных данных из Библиотеки Паладинов.
- Локальное хранение посещений, зачистки и собранного дропа в браузере.
- Адаптивная панель, которая старается не перекрывать игровое окно целиком.

## Поддерживаемые подземелья

- Грибница: 1-4 этажи.
- ППК: 1-8 этажи.

Список подземелий расширяется через `src/dungeons.js` и данные карт в `data/maps/`.

## Установка из GitHub

1. Открой страницу официального репозитория проекта.
2. Перейди в раздел `Releases`.
3. Скачай последнюю версию расширения.
4. Распакуй папку расширения в удобное место.
5. Открой в браузере `chrome://extensions`.
6. Включи `Режим разработчика`.
7. Нажми `Загрузить распакованное расширение`.
8. Выбери распакованную папку с `manifest.json`.
9. Зайди в подземелье на `combats.com`: справа появится панель карты.

## Обновление

1. Скачай новую версию из `GitHub Releases`.
2. Замени старую папку расширения новой.
3. Открой `chrome://extensions`.
4. Нажми `Обновить` у расширения.
5. Перезагрузи вкладку игры.

Перед обновлением лучше закрыть вкладку с подземельем, чтобы браузер не держал старый контекст расширения.

## Как пользоваться

- Клик по клетке показывает информацию и строит маршрут.
- Клик по монстру, NPC, сундуку или тайнику в боковой панели строит маршрут к нему.
- Кнопка `Игрок` приближает карту к текущему персонажу.
- Кнопка `Вся карта` показывает весь этаж.
- Карта двигается зажатой левой кнопкой мыши.
- Посещенные клетки получают красную рамку.
- Убитые монстры отмечаются как зачищенные.
- Вкладку с объектами можно скрывать и открывать кнопкой слева на карте.

## Производительность и приватность

Расширение работает локально в браузере. Оно не отправляет игровые данные на сторонние серверы автора. Данные карт и справочников берутся с `lib.paladins.ru`, игровые координаты читаются из текущей страницы игры, а состояние карты сохраняется в `chrome.storage.local`.

Расширение не заменяет игровой клиент и не выполняет игровые действия за игрока. Оно только отображает карту, справочную информацию и маршруты.
